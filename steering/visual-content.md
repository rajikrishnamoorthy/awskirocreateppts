# Adding Visual Content to Presentations

Follow this guidance when adding images, charts, tables, and shapes to your slides.

## Images

### When to Use Images
- Illustrate concepts or ideas
- Add visual interest
- Show products, people, or places
- Break up text-heavy slides

### Best Practices
- Use high-resolution images (minimum 72 DPI for screen, 300 DPI for print)
- Maintain aspect ratio when resizing
- Position images consistently across slides
- Use `add_image` with local file paths or URLs

### Supported Formats
- PNG (recommended for graphics with transparency)
- JPEG (recommended for photographs)
- GIF, BMP, TIFF

## Charts

### Chart Types and When to Use Them
- **Bar/Column charts** - Comparing categories
- **Line charts** - Showing trends over time
- **Pie charts** - Showing proportions (use for 3-6 segments max)
- **Area charts** - Cumulative trends
- **Scatter plots** - Correlations between variables

### Chart Best Practices
- Include clear titles and labels
- Use contrasting colors for data series
- Limit data points for clarity
- Add data labels when helpful

## Tables

### When to Use Tables
- Comparing multiple items across attributes
- Presenting structured data
- Showing schedules or timelines
- Listing features or specifications

### Table Best Practices
- Use header row with bold/contrasting style
- Align numbers to the right
- Keep columns to essential data only
- Use alternating row colors for readability

## Shapes

### Common Shape Uses
- Flowcharts and process diagrams
- Callouts and annotations
- Decorative elements
- Grouping related content

### Shape Styling
- Use consistent fill colors
- Apply subtle shadows for depth
- Maintain uniform border styles
- Group related shapes together
